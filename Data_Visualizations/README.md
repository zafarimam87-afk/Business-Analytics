# Data Visualizations
This directory contains projects focused on transforming complex datasets into actionable insights using visual storytelling.

### Tools Used
* **Tableau / Power BI:** Interactive dashboards and executive summaries.
* **Python (Matplotlib/Seaborn):** Exploratory data analysis and custom statistical plots.

### Featured Projects
    * **[Excel Interactive Dashboard](DataVisualization%20in%20Excel.xlsx)**: A comprehensive data visualization project utilizing advanced PivotTables, dynamic charting, and conditional formatting to distill complex business data into trackable KPIs.
    * **[Tableau Sales Dashboard](Tableau_Sales_Dashboard.twbx)**: An interactive Tableau workbook showcasing sales performance metrics, regional trends, and profitability analysis.
    * **[S&P 500 Market Cap Analysis](Tableau_%20S%26P%20500%202008-'15%20Market%20Caps_VIZ.twbx)**: A longitudinal visualization of S&P 500 market capitalizations from 2008-2015.
    * **[Power BI Market Analysis](Power_BI_Project.pbix)**: A detailed Power BI report focusing on market trends and data-driven decision making (accompanied by a [PDF Summary](PowerBI_Project.pdf)).
    * **[Global Health & Economics (Gapminder)](Tableau_Gapminder%20Project.twbx)**: A Tableau project inspired by Hans Rosling's Gapminder, visualizing the relationship between health and wealth over time.
    * **[Airline Performance Analysis](Best%20Airline.jpg)**: A visual analysis (infographic/chart) evaluating airline performance metrics.
